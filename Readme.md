birdbgp
=======

NodeJS interface to the Bird BGP daemon

Requirements
------------

* NodeJS
* Bird running locally

Installation
------------

With NPM in your project directory:

	npm install birdbgp

Using GIT inside your `node_modules` directory:

	git clone https://www.github.com/jeffwalter/birdbgp.git
